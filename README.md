# University
Study project about university.

Each university has next fields:
 - Name;
 - Address;
 - Rating;
 - Presence of foreign branches;
 - Type;
 - Number of professors;
 - Foundation date;
 - Faculties.

Each faculty has next fields:
 - Name;
 - Educational path;
 - Dean's name;
 - Foundation date;
 - Budget;
 - Number of subjects.
